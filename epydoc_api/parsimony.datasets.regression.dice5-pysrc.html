 73      Details 
 74      ------- 
 75      The general procedure is: 
 76          1) For each pixel i, Generate independant variables Xi ~ N(0, 1) 
 77          2) Add object level structure corresponding to the five dots: 
 78             - Sample five latent variables ~ N(0, 1): l1, l12, l3, l4, l45. 
 79             - Pixel i of dots X1, X2, X3, X4, X5 are sampled as: 
 80               X1i = l1 + l12 + Xi 
 81               X2i = l12 + Xi 
 82               X3i = 2 * l3 + Xi 
 83               X4i = l4 + l45 + Xi 
 84               X5i = l45 + Xi 
 85               Note that: 
 86               Pixels of dot X1 share a common variance that stem from l1 + l12. 
 87               Pixels of dot X2 share a common variance that stem from l12. 
 88               Pixels of dot X1 and pixel of dot X2 share a common variance that 
 89               stem from l12. 
 90               etc. 
 91          4) Spatial Smoothing. 
 92          5) Model: y = X beta + noise 
 93          - Betas are null outside dots, and 1 or -1 depending on the dot: 
 94              X1: 1, X2: -1, X3: 1, X4: 1, X5: -1. 
 95          - Sample noise ~ N(0, 1) 
 96          - Compute X beta then scale beta such that: r_squared(y, X beta) = r2 
 97          Return X, y, beta
