  2  """ 
  3  The :mod:`parsimony.algorithms.implicit` module includes several algorithms 
  4  that minimises an implicit loss function. 
  5   
  6  Algorithms may not store states. I.e., if they are classes, do not keep 
  7  references to objects with state in the algorithm objects. It should be 
  8  possible to copy and share algorithms between e.g. estimators, and thus they 
  9  should not depend on any state. 
 10   
 11  There are currently two main types of algorithms: implicit and explicit. The 
 12  difference is whether they run directly on the data (implicit) or if they have 
 13  an actual loss function than is minimised (explicit). Implicit algorithms take 
 14  the data as input, and then run on the data. Explicit algorithms take a loss 
 15  function and possibly a start vector as input, and then minimise the function 
 16  value starting from the point of the start vector. 
 17   
 18  Algorithms that don't fit well in either category should go in utils instead.

 65   
 66          Examples 
 67          -------- 
 68          >>> import numpy as np

 70          >>> 
 71          >>> np.random.seed(0) 
 72          >>> X = np.random.random((10, 10)) 
 73          >>> fast_svd = FastSVD() 
 74          >>> fast_svd.run(X)

 76                 [-0.35647707], 
 77                 [-0.35190104], 
 78                 [-0.34715338], 
 79                 [-0.19594198], 
 80                 [-0.24103104], 
 81                 [-0.25578904], 
 82                 [-0.29501092], 
 83                 [-0.42311297], 
 84                 [-0.27656382]]) 
 85          >>> 
 86          >>> np.random.seed(0) 
 87          >>> X = np.random.random((100, 150)) 
 88          >>> fast_svd = FastSVD() 
 89          >>> v = fast_svd.run(X) 
 90          >>> us = np.linalg.norm(np.dot(X, v)) 
 91          >>> s = np.linalg.svd(X, full_matrices=False, compute_uv=False) 
 92          >>> abs(np.sum(us ** 2.0) - np.max(s) ** 2.0) 
 93          9.0949470177292824e-13 
 94          >>>

 96          >>> X = np.random.random((100, 50))

 98          >>> v = fast_svd.run(X) 
 99          >>> us = np.linalg.norm(np.dot(X, v)) 
100          >>> s = np.linalg.svd(X, full_matrices=False, compute_uv=False) 
101          >>> abs(np.sum(us ** 2.0) - np.max(s) ** 2.0)

106          M, N = X.shape 
107          if M < 80 and N < 80:  # Very arbitrary threshold for my computer ;-) 
108              _, _, V = np.linalg.svd(X, full_matrices=True) 
109              v = V[[0], :].T 
110          elif M < N: 
111              K = np.dot(X, X.T) 
112              t = start_vector.get_vector(X.shape[0]) 
113              for it in xrange(max_iter): 
114                  t_ = t 
115                  t = np.dot(K, t_)
